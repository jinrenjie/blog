[Outline](https://getoutline.org/en/home) 是基于 [Shadowsocks](https://shadowsocks.org/en/index.html) 实现的一套开源软件，但是其部署的友好程度超乎想象，如果你用了 Outline Manager 内推荐的 VPS 服务商，只需按照指引步骤一步步执行即可完成部署，相当方便！值得一提的是截止目前为止，虽然有不少用户在 Github 仓库提了需求，但该团队仍然没有考虑加入 PAC（Porxy Auto Config）功能。

## 所需工具
* Outline Manager
* Outline Client
* VPS

## Outline Manager

Outline Manager 支持 [Windows](https://raw.githubusercontent.com/Jigsaw-Code/outline-releases/master/manager/stable/Outline-Manager.exe)、[Linux](https://raw.githubusercontent.com/Jigsaw-Code/outline-releases/master/manager/stable/Outline-Manager.AppImage) 和 [Mac OS](https://raw.githubusercontent.com/Jigsaw-Code/outline-releases/master/manager/stable/Outline-Manager.dmg)，现在后直接运行即可。

![初始界面](/images/article/outline-manager-index.png)

可以看到默认支持四种服务商，这里我选用的是 [DigitalOcean](https://cloud.digitalocean.com)，如果你已经有账户了，可以直接点击 SET UP 进行 登录，如果没有直接注册。

![登录成功](/images/article/outline-manager-sign-in.png)
当 Outline Manager 获取用户信息后，你可以点击 CREATE SERVER 创建你的服务器，

![选择地区](/images/article/outline-manager-select-location.png)
选择一个地区，并点击右上角的 SET UP OUTLINE

![等待自动创建](/images/article/outline-manager-createing-server.png)
等待 Outline Manager 自动创建服务器并部署服务
